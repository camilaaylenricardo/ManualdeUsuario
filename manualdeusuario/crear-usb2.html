Crear USB Booteable

⚠️ ¡Advertencia! Al crear un USB booteable, todos los datos en la unidad USB serán borrados. Asegúrate de hacer una copia de seguridad de cualquier archivo importante antes de continuar. También es importante descargar el archivo ISO de Windows desde una fuente confiable (como el sitio oficial de Microsoft) para evitar la instalación de software corrupto o malicioso.

Para instalar Windows, necesitas crear un USB booteable. Te recomendamos utilizar Rufus:

  1. Descarga la imagen ISO de Windows:
    • Windows 10
      • SI no sabes como hacerlo, Haz clic aqui para un ejemplo
  2. Descarga Rufus y ejecútalo.
  3. Conecta la unidad USB (mínimo 8 GB).
  4. Selecciona la unidad USB y elige la imagen ISO que descargaste.
  5. Selecciona MBR o GPT según tu PC.
  6. Haz clic en "Iniciar" y sigue las instrucciones para completar el proceso.
