Instalación de Windows 10

Para instalar Windows 10, asegúrate de que tu PC cumpla con los siguientes requisitos mínimos:

  • Procesador de 1 GHz
  • RAM: 1 GB para 32 bits o 2 GB para 64 bits
  • Disco duro: 16 GB para 32 bits o 20 GB para 64 bits
  • Pantalla de 800x600 y tarjeta gráfica
